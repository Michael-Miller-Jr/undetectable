Show Me How
How to find Class or ID selectors:

1. Right-click on the annoying ad or popup
2. Choose "Inspect" to open DevTools
3. Hover over elements until the right one highlights, and note the name in between the " marks
4. Look for a Class (e.g. <div class="promo-banner">) or ID (e.g. <div id="ad-overlay">)
5. Copy and paste the Class or ID name (aka selector) into the appropriate box below



Tip: Classes start with . and IDs start with #

Example Class:
<div class="promo-banner"> should be entered as .promo-banner

Example ID:
<div id="ad-overlay"> should be entered as #ad-overlay

*If you make a mistake, remove the last . or # added
